# CMVideo

A media player written in C# using LibVLCSharp and WPF, constantly being updated with new features.



 # Features  

 - [x] Pause/Play
 - [x]  Volume control
 - [x] Forward and backward direct seeking
 - [x] Precision Seeking with seekbar
 - [x] Playlisting
